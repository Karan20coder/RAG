<h2>🚀 About The Project</h2>

<p>
PDF RAG System is a full-stack AI application that enables users to upload text-based PDF documents and ask natural language questions using advanced semantic search. The entire system runs 100% locally with no API keys, no subscriptions, and no third-party servers. It ensures complete privacy while delivering intelligent, context-aware answers extracted directly from your documents.
</p>

<hr style="border: 1px solid #f6d6c9;"/>

<h2>🧠 Core Features</h2>

<ul>
  <li>📂 Drag & Drop PDF Upload</li>
  <li>🧠 Natural Language Question Answering</li>
  <li>⚡ Semantic Search (Not Keyword Based)</li>
  <li>🔍 Relevance Scoring</li>
  <li>🔒 Fully Local AI Processing</li>
  <li>📶 Works Offline After Setup</li>
</ul>

<hr style="border: 1px solid #f6d6c9;"/>

<h2>⚙️ Architecture Flow</h2>

<pre>
PDF → Text Extraction → Smart Chunking → Embeddings
→ FAISS Vector Index → Semantic Similarity Search
→ Context Retrieval → Intelligent Response
</pre>

<p>
Instead of traditional keyword matching, this system converts both document text and user queries into high-dimensional vector embeddings. It then performs similarity search using FAISS to retrieve the most contextually relevant sections.
</p>
